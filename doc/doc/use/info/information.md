
# 核心信息面板
当您点击屏幕左下角红色的 <el-button type="danger" circle>^_^</el-button> (版本信息)
按钮时，屏幕会弹出一个面板，面板包含【标准产品相关记录】、【运维管理】、【项目信息】、【子应用信息】及【网址收藏】几个按钮

![image.png](/info/1.png)

::: danger 注意事项：
- 按钮拖动可以自定义展示顺序。
- 对于普通网页，只会展示【标准产品相关记录】及【网址收藏】两个按钮。
:::


## 标准产品相关记录
每当您成功登录一个标准产品网址，脚本会自动将网址信息进行记录。点击【标准产品相关记录】即可查看当前记录的标准产品网址信息

![image.png](/info/2.png)
点击【展开】按钮，可以展示更多关于该网址的信息面板。
![image.png](/info/3.png)
点击【新增】，可以输入与当前网址相关的内容，比如该网址的ux设计图地址，接口文档链接等内容。<br />
点击【下载】，会将当前网址的所有信息以表格的方式进行下载，便于储存。
![image.png](/info/4.png)
::: tip 提示
- 点击网址链接，会自动进行跳转。
- 输入账号和密码时，使用“/”隔开，脚本会自动识别。
- 点击相关文字，脚本会自动将其复制至您的剪切板。
:::


## 运维管理
点击【运维管理】，即可进入系统运维管理相关界面。

![image.png](/info/5.png)

运维管理界面相当于系统运维平台的一个快捷入口，包含几乎运维平台的所有功能。

![image.png](/info/6.png)

您在此处的所有操作相当于直接在运维平台进行操作，因此：
<b style="color:red">为了数据安全，我们强烈建议您不要轻易进行账号、权限相关操作！</b>



::: danger 注意：
- 对于标准产品4.3版本以下的任何界面，您都可以打开【运维管理】界面，即使您在登录界面还未登录。
- 对于标准产品4.3以上版本，只有成功登录后才能使用此功能！
:::

## 项目信息
点击【项目信息】后，会展示当前系统的项目信息、用户登录信息等基础内容。这在您想知道该系统的所有项目名称、项目id时会非常有用！
![image.png](/info/7.png)
::: danger 注意：
 - 点击信息文本，可以将其复制至剪切板。
 - 对于标准产品4.3以上版本，只有成功登录后才能使用此功能！
:::

## 子应用信息
点击【子应用信息】，会自动生成菜单栏的框架。

![image.png](/info/9.png)

点击一级菜单栏，会显示其内所有子应用的链接、挂接方式等信息。

![image.png](/info/10.png)

::: info 
在标准产品4.3版本以上，子应用的挂接方式均采用乾坤方案，此时，您无法通过浏览器的调试面板获取到其正式链接（iframe挂接的方式可以获取到）
但使用此功能，您依旧可以方便的进行子应用真实链接查看。
:::


点击【搜索】可以快速查找一个子应用，支持URL、服务名、中文关键词等搜索。

![image.png](/info/info11.png)
<br />点击【下载】，会将当前菜单的所有具体信息输出，如下图。

![image.png](/info/8.png)
